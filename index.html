เกี่ยวกับฉัน

ผมเป็นนักศึกษาที่มีความกระตือรือร้นในการเรียนรู้เทคโนโลยีใหม่ๆ และมีความสนใจในการออกแบบ User Experience และ User Interface พร้อมทั้งมีทักษะในการเขียนโปรแกรมหลายภาษา

ผมกำลังศึกษาอยู่ที่ มหาวิทยาลัยเทคโนโลยีราชมงคลธัญบุรี ในสาขาวิทยาการคอมพิวเตอร์ ซึ่งทำให้ผมได้มีโอกาสเรียนรู้ทั้งทฤษฎี และปฏิบัติจริงในการพัฒนาซอฟต์แวร์และการออกแบบ

ในด้านการทำงาน ผมมีทักษะในการทำงานเป็นทีม มีความเป็นผู้นำ และสามารถแก้ไขปัญหาได้ดี พร้อมทั้งบริหารเวลาได้อย่างมีประสิทธิภาพ
